# 13 Object-Relational Mapping (ORM): E-Commerce Back End

## Description
This project involves building the back end for an e-commerce site using Express.js API, Sequelize, and a MySQL database. It demonstrates the fundamental architecture of e-commerce platforms and provides an example of how to interact with a database using Sequelize ORM. This application includes models for categories, products, and tags, as well as the associations between them.
